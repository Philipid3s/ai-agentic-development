# Scope v1

## In Scope

- Standardized agent registry and role files
- Skill packages for financial advisory and security audit
- Orchestration docs (workflows + handoffs)
- Ops docs (runbooks + monitoring)

## Out of Scope

- Fully autonomous execution for high-risk actions
- Detailed legal or regulated financial advisory logic
- Organization-wide IAM or enterprise governance implementation

## v1 Exit Criteria

- Team can onboard a new role or skill using templates only
- At least one end-to-end workflow is documented and reproducible
- Risk escalation path is documented and testable
